- model context protocol server for enzyme
- allows enzyme to be a context provider for other apps
- functionality:
  - retrieve notes from obsidian vault using tag, link, folder

- for example, you can ask:
  - 50 notes tagged with #writing
  - 25 notes from my Readwise/Books folder

- it will grab latest notes from your vault
- something that cursor doesn't do yet (because it just searches for mention)
- plans for more sophisticated understanding of contents in the future to help it more effectively use them, even automatically
